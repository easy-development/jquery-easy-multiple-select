Getting started

1. Include the Assets

        <script src="em-select/js/jquery.js"></script>

        <link href="em-select/css/style.css" rel="stylesheet" type="text/css">
        <script src="em-select/js/jquery-em-select.min.js"></script>

2. Trigger jQuery Easy Multiple Select

2.1 By jQuery

            jQuery(".selectIdentifier").em_select();

2.2 By Adding a class to your select

Add the following class to the selectors you want to work like this : component-em-select

3. Enjoy
